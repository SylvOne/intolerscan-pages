Politique de Confidentialité

Dernière mise à jour : 30 octobre 2025

1. Introduction

IntolerScan est une application mobile qui vous aide à identifier les aliments susceptibles de causer des intolérances alimentaires. Nous prenons très au sérieux la protection de vos données personnelles et de santé.

2. Données Collectées

Nous collectons les types de données suivants :

  • Données d'identification : Adresse email (pour l'authentification via Firebase)
  • Données de santé : Symptômes, repas consommés, ingrédients scannés, corrélations alimentaires
  • Données d'utilisation : Scans de produits, exports PDF, utilisation des fonctionnalités
  • Données d'achat : Informations sur vos abonnements Premium (via RevenueCat)
  • Photos : Images de produits que vous scannez (stockées via Firebase Storage)

3. Utilisation des Données

Vos données sont utilisées uniquement pour :

  • Vous fournir les fonctionnalités de l'application (suivi, analyses, corrélations)
  • Améliorer nos algorithmes de détection d'intolérances
  • Gérer votre compte et vos abonnements
  • Vous fournir un support technique

4. Partage des Données

Nous ne vendons jamais vos données. Nous partageons certaines données uniquement avec :

  • Firebase (Google) : Pour l'authentification, le stockage cloud et la base de données
  • RevenueCat : Pour la gestion des abonnements et achats in-app
  • OpenFoodFacts API : Pour récupérer les informations nutritionnelles des produits

Ces services tiers sont conformes au RGPD et protègent vos données.

5. Sécurité

Vos données sont :

  • Chiffrées en transit (HTTPS/TLS)
  • Stockées sur des serveurs sécurisés Firebase
  • Protégées par authentification Firebase
  • Accessibles uniquement par vous

6. Vos Droits (RGPD)

Conformément au RGPD, vous disposez des droits suivants :

  • Droit d'accès : Vous pouvez demander une copie de toutes vos données
  • Droit de rectification : Vous pouvez corriger vos données dans l'app
  • Droit à l'effacement : Vous pouvez supprimer votre compte et toutes vos données
  • Droit d'opposition : Vous pouvez vous opposer au traitement de vos données
  • Droit à la portabilité : Vous pouvez exporter vos données (via export PDF)

7. Conservation des Données

Nous conservons vos données tant que votre compte est actif. Si vous supprimez votre compte, toutes vos données sont effacées sous 30 jours maximum.

8. Données de Santé

IntolerScan collecte des données de santé sensibles (symptômes, intolérances). Ces données :

  • Ne sont jamais partagées avec des tiers à des fins commerciales
  • Ne sont utilisées que pour vos analyses personnelles
  • Restent sous votre contrôle total

9. Cookies et Technologies de Suivi

L'application n'utilise pas de cookies web. Les données d'utilisation sont collectées uniquement pour améliorer votre expérience.

10. Modifications de cette Politique

Nous pouvons modifier cette politique de confidentialité. Toute modification sera communiquée via l'application et prendra effet 30 jours après notification.
